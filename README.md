# Blackjack Botsons
This is a Discord bot mainly made for entertainment and education.

## Commands
* bjdeal: Deals cards.
* bjhelp: Sends the help embed.
* bjinvite: Sends a link to bots homepage.
* bjjoin: Join an ongoing game.

## Controls

* Click ✅ to deal a card.
* Click ❌ to stay.
* Click 💳 to double down.

### Bets

* 💵: 100$
* 💰: 500$
* 💎: 1000$
